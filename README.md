# emailDomainValidator.jquery.js
Library to validate email domain in client-side email fields with jQuery


## Usage
```javascript
$('.email-validator').emailDomainValidator([
  'gmail.com'
], function() {
  console.log('success');
}, function() {
  console.log('failed');
});
```
